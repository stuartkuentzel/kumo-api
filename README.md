## Description

[kumo.chat](https://kumo.chat) chat application. Rent chat rooms anyonymous that self destruct in 24 hours, using the Lightning Network for payments. Built ontop of the Nest Framework.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
